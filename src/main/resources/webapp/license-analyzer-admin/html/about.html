Polarion User License Analyzer

User license analyzer for Polarion ALM. Parse log files and save statistics about users and license types. Provide Open API and Rest API functions.

Recommendation

Polarion version - 2304+

JDK version - 17.0.11

RAM - 6Gb+

Java heap space - 2GB+
